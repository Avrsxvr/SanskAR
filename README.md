Overview

SanskritAR is a mobile augmented reality application designed for interactive exploration of cultural heritage content. The app uses real-world placement and immersive storytelling to enhance learning and engagement.

Features
 1. Real-world object placement using AR Foundation
 2. Interactive storytelling elements
 3. Optimized tracking, scale accuracy, and rendering for mobile AR

Tech Stack
 1. Unity
 2. AR Foundation
 3. ARCore / ARKit
 4. C#

Setup
 1. Open the project in Unity
 2. Install AR Foundation and platform-specific AR packages
 3. Switch build platform to Android or iOS
 4. Deploy to a supported mobile device
